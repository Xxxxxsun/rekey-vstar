A contamination signal in static V*Bench

We first ask whether current VQA benchmarks are still measuring visual perception, or whether part of the score comes from memorized answer-bearing details. As a probe, we keep the original images fixed but replace the question with a fresh human-written question targeting a different visual key. If a model truly solves the image, performance should remain stable; instead, accuracy drops sharply across model families.

This consistent drop indicates that original V*Bench scores are partly inflated by memorized answer-bearing details. REKEY starts from this observation and makes the visual key renewable.

How REKEY Renews a Visual Key

REKEY does not only rewrite the question. It identifies editable visual-key slots, regenerates the local visual evidence under context, and then writes a fresh question whose answer depends on the newly generated key.

Step 1

Annotate renewable slots

Starting from the original VQA image, REKEY marks candidate edit slots and records what each slot can support, such as adding a cup or changing the color of an existing object.

Step 2

Generate within context

REKEY crops a larger context window around the slot, prompts a generative model to edit only the marked region, and keeps the surrounding pixels fixed.

Step 3

Paste back and re-question

The generated patch is pasted back into the full image, producing a refreshed benchmark item with a new answer-bearing visual key and a matching multiple-choice question.

Experimental Results

We ask three questions: does REKEY close the contamination gap (RQ1), does it preserve V*Bench's original difficulty (RQ2), and is human annotation necessary (RQ3)?

RQ2

REKEY preserves the original difficulty

On LLaVA-1.5-13B, position accuracy barely moves (−0.4 pp) after regeneration — REKEY preserves the visual-search structure while renewing the answer-bearing evidence.

RQ3

Human annotation is necessary

GPT-5.5's self-annotations put the new object in add regions that are 3.2× larger than human ones, making the benchmark +8.4 to +18.5 pp easier across models — human curation is what keeps the difficulty honest.

Robustness

Capability boosts that disappear under REKEY

  • Chain-of-thought lifts Original / Relabel by up to +6.8 / +10.5 pp — but on REKEY the effect turns slightly negative (−0.5 to −3.1 pp).
  • Agentic zoom adds up to +9.5 / +12.4 pp on the static sets, yet only −4.0 to +3.6 pp on REKEY.
